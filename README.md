# cheapo-streamdeck-thing
It's just a few buttons and an LCD, made for my delightful wife :)


Uses an aliexpress ssd1306-based OLED, RP2040 Zero and some switches and LEDs. 



Images **MUST** be in 1 bit bmp format!
